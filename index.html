Claim control of your Docker images!

What is Portus

Portus is an open source authorization service and user interface for the next generation of Docker Registry.

Why using Portus

The Docker Hub is a wonderful place, but what can you do if you cannot or don't want to push your data to it? You can setup your private instance of Docker registry but then everybody inside of your organization will be able to push and pull privileges over all your images. Also how are you going to keep track of all the awesome images that have been pushed to your private registry?

Portus to the rescue!

Secure

Portus implements the new authorization scheme defined by the latest version of the Docker registry. It allows to have fine grained control over all your images deciding who can pull and push them.

Easily manage users within teams

Map your company organization inside of Portus, define as many teams as you want and add and remove users from them.

Teams have three different types of users to allow full granularity:

  • Viewers: can only pull images.
  • Contributors: can push and pull images
  • Owners: like contributos, but can also add and remove users to the team.

Search

Portus provides an intuitive overview of the contents of your private registry. It also features a search capability to find images even faster.

User privileges are constantly taken into account, even when browsing the contents of the repository or when performing searches.

Audit

Keep everything under control. All the relevant events are automatically logged by Portus and are available for consultation to the admin users.

Non-admin users can use this feature to be keep up with relevant changes.

Try Portus
